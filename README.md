# coffee-sales-dashboard-excel

## Project Overview
This project analyzes coffee sales data using Microsoft Excel to uncover sales trends, customer behavior, and product performance through an interactive dashboard.

## Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- XLOOKUP
- INDEX-MATCH
- Slicers
- Timeline
- Conditional Formatting

## Key Insights

- Highest revenue-generating coffee type
- Monthly sales trends
- Regional sales comparison
- Top customers
- Country-wise performance

- CoffeeSalesDashboard.xlsx

- CoffeeSalesData(raw data)

- Data Cleaning
